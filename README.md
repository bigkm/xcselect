Basic usage

    $ xcselect 
    Please Select an Xcode install ?
    
       [1]    Xcode 4.1 (4B110) /Developer 
       [2]    Xcode 4.2 (4D5163b) /Xcode42 [current]
    
    Selection: 


Use -a to switch to the next xcode  

    xcselect -a
    
use show -s to print info about the current xcode  

    xcselect -s
